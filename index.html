⚙️ How to set up the calendar ▾
Step 1 — Get your Calendar IDs
1
Open Google Calendar on a computer (not phone)
2
Find the calendar you want to add in the left sidebar (e.g. School or Family)
3
Click the three dots next to its name → Settings and sharing
4
Scroll down to Integrate calendar → copy the Calendar ID (looks like an email address)
5
Repeat for each calendar you want to show
Step 2 — Build your embed URL
1
Start with this base URL:
https://calendar.google.com/calendar/embed?
2
Add each calendar ID like this (one &src= per calendar):
&src=CALENDAR_ID_1&src=CALENDAR_ID_2
3
Add display settings at the end:
&ctz=America%2FNew_York&showTitle=0&showNav=1&showDate=1&mode=AGENDA
4
Your finished URL will look like:
https://calendar.google.com/calendar/embed?src=abc123%40gmail.com&src=xyz456%40group.calendar.google.com&ctz=America%2FNew_York&showTitle=0&mode=AGENDA
Step 3 — Replace the placeholder
1
Open this HTML file in a text editor (on Chromebook: right-click → Open with → Text)
2
Find the comment that says SWAP THIS BLOCK
3
Delete the entire <div class="cal-placeholder">...</div> block
4
Replace it with:
<iframe src="YOUR_URL_HERE" style="width:100%;height:300px;border:none;display:block" frameborder="0"></iframe>
5
Save the file and re-upload to Google Drive
To add or remove a calendar later
+
Add: Find the new calendar's ID (Step 1 above), then add &src=NEW_ID to your existing iframe URL
−
Remove: Delete the &src=THAT_CALENDAR_ID portion from the iframe URL and save
💡 Google Classroom tip: When you connect Classroom to Google Calendar (Settings → Integrate Calendar), a new calendar called something like "Class: Biology" appears automatically. Add its ID here the same way — all assignments will show up on this page.
